# Place your phpMyAdmin CSV exports here

This directory is read by the migration scripts in `scripts/migrate/`.

## Required CSV files

| File                     | Content                            |
|--------------------------|------------------------------------|
| `specializations.csv`    | Doctor specializations             |
| `doctors.csv`            | Doctor records                     |
| `patients.csv`           | Patient records                    |
| `appointments.csv`       | Appointment records                |
| `invoices.csv`           | Invoice headers                    |
| `invoice_items.csv`      | Invoice line items (optional)      |
| `medicines.csv`          | Medicine catalog                   |
| `inventory.csv`          | Inventory items                    |
| `staff.csv`              | Staff / user accounts              |
| `prescriptions.csv`      | Prescription headers               |
| `prescription_items.csv` | Prescription medications (optional)|
| `lab_results.csv`        | Lab result records                 |

## How to export from phpMyAdmin

1. Open phpMyAdmin → select your database
2. Click on a table → **Export** tab
3. Format: **CSV**
4. Check **Put column names in the first row**
5. Click **Go** → save file to this folder with the name shown above

## id-maps/

Auto-generated by migration scripts. Maps old MySQL integer IDs to new MongoDB ObjectIds.
Do not edit manually.
