# Dashboard

The Dashboard is the first screen you see after logging in. It provides a real-time summary of clinic activity tailored to your role.

---

## Selecting a Time Period

At the top of the dashboard, use the **period selector** to filter metrics:

| Option | Description |
|---|---|
| **Today** | Activity for the current calendar day |
| **Week** | Activity for the current week |
| **Month** | Activity for the current month |

All metrics — appointment counts, revenue, and patient stats — update instantly when you change the period.

---

## Admin / Owner Dashboard

Admins and owners see the broadest view of clinic performance.

### Key Metrics

| Metric | Description |
|---|---|
| **Total Patients** | All registered patients in the system |
| **Total Doctors** | Active doctors on the system |
| **Today's Appointments** | Appointments scheduled for today |
| **Period Appointments** | Appointments in the selected period |
| **Period Visits** | Clinical visits completed in the period |
| **Period Revenue** | Total billed amount in the period |
| **Outstanding Invoices** | Unpaid or partially paid invoice total |

### Upcoming Appointments
A list of the next scheduled appointments showing:
- Appointment code
- Patient name
- Assigned doctor
- Date, time, and status

### Payment Breakdown
A summary of payment methods used in the period (cash, card, HMO, etc.) and the amounts collected per method.

---

## Doctor Dashboard

Doctors see a focused view on their own patient load.

### Key Metrics
- **My Today's Appointments** — appointments assigned to you today
- **My Period Appointments** — your appointments in the selected period
- **My Completed Visits** — visits you have finished in the period
- **My Pending Prescriptions** — prescriptions you have issued that are pending dispensing

### My Schedule Today
A chronological list of your appointments for the day with patient names, times, and statuses.

### Recent Visits
Your most recently documented clinical visits with quick links to open each record.

---

## Nurse Dashboard

Nurses see the current queue and patient care activity.

### Key Metrics
- **Patients in Queue** — patients currently waiting
- **Vitals Recorded Today** — number of patients you have triaged
- **Completed Visits Today** — visits marked as done today

### Queue Overview
A live snapshot of the patient queue showing queue number, patient name, status, and assigned doctor.

---

## Receptionist Dashboard

Receptionists see appointment and registration activity.

### Key Metrics
- **Today's Appointments** — all appointments for today
- **Walk-ins Today** — walk-in patients added to the queue today
- **Pending Invoices** — invoices not yet fully paid

### Today's Appointment List
Full list of today's appointments with patient names, doctors, times, and statuses. You can click any row to open the appointment detail.

---

## Accountant Dashboard

Accountants see a financial summary.

### Key Metrics
- **Period Revenue** — total billed amount
- **Amount Collected** — payments received
- **Outstanding Balance** — unpaid invoices
- **Invoices Issued** — total invoices created in the period

### Outstanding Invoices
A list of invoices with balances due, showing patient name, invoice amount, due date, and status.

---

## Tips

- The dashboard auto-refreshes. You can also reload the page manually to get the latest data.
- Clicking a patient name, appointment code, or invoice on the dashboard navigates directly to that record.
- If a metric shows zero unexpectedly, check that the selected time period is correct.
